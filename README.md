Increases the height of the experience bar, in order to fit a label showing the player's current and required experience to level up, and a 2DP percentage of their progress through the level.

This mod adds the following configuration options to MelonPreferences.cfg:
DisableTicks (default true) - Whether to disable the ticks at every 10% on the xp bar. Makes the label easier to read.
FontSize (default 16) - The font size to set for the label on the xp bar.
ShowPercentGain (default true) - Adds the percentage of xp gained to the xp gained chat messages)
ShowXpGain (default true) - Replaces "some experience" with the exact experience gained in xp gain chat messages
